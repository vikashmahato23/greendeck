## Deployment

To deploy this project run

### clone It
```
 https://github.com/vikashmahato23/greendeck.git
```
### backend node_modles install
```
 yarn install
```
### backend run
```
 yarn start
```
### frontend
```
cd frontend
```
### React.js install
```
yarn install
```
### frontend run
```
yarn start
```
**note** :- react should run in port 3000 and nodejs will run by defalut 5000
